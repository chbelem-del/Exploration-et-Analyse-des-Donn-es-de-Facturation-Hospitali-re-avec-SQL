# Analyse de Données Hospitalières | SQL Server

## Présentation

Ce projet consiste à analyser une base de données hospitalière composée d'une table de faits et de plusieurs tables de dimensions. L'objectif est d'explorer le parcours complet d'un patient, depuis les soins jusqu'à la facturation, les paiements des assurances et les ajustements financiers, afin de répondre à des problématiques métier à l'aide de SQL.

---

## Objectifs

- Créer et structurer une base de données relationnelle.
- Importer et vérifier les données.
- Explorer les données avec SQL.
- Réaliser des jointures et des agrégations.
- Répondre à des questions métier à travers des requêtes SQL.

---

## Technologies utilisées

- SQL Server
- SQL

---

## Analyses réalisées

- Création du schéma relationnel
- Importation des données
- Définition des clés primaires et étrangères
- Jointures entre la table de faits et les dimensions
- Analyses des patients, médecins, diagnostics, paiements et ajustements
- Requêtes d'agrégation et calculs métier

---

## Fonctionnalités

- Modélisation relationnelle
- Requêtes SQL (SELECT, JOIN, GROUP BY, HAVING)
- Fonctions d'agrégation
- Analyse des flux financiers hospitaliers
- Résolution de problématiques métier
